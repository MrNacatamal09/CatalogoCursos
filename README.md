# CatalogoCursos
Pequeña demo en C# que permite buscar cursos por nombre desde la consola.

Este es un pequeño programa de consola en C# que implementa un catálogo de cursos con una función básica de búsqueda por nombre.
El usuario puede escribir una palabra o parte del nombre de un curso, y el programa mostrará todos los cursos que coincidan sin importar mayúsculas o minúsculas.
El proyecto demuestra el uso de:
Colecciones genéricas (List<>)
Tuplas para representar datos estructurados
LINQ para realizar filtros y consultas
Entrada y salida en consola

Ideal como ejemplo simple de filtrado de datos en C# o como base para un sistema de gestión más completo.

MANUAL DE USUARIO:
Ejecuta el programa desde tu terminal o entorno de desarrollo (por ejemplo, Visual Studio o dotnet run).

En pantalla verás el título:
Catálogo de Cursos - Demo
Buscar:
Escribe una palabra o parte del nombre del curso que deseas buscar y presiona Enter.
La búsqueda no distingue mayúsculas o minúsculas.
El programa mostrará todos los cursos que coincidan con tu búsqueda.
Ejemplo:
Buscar:
prog
[2] Introducción a la progrmación - CS
Si no escribes nada y presionas Enter, se mostrarán todos los cursos disponibles.
